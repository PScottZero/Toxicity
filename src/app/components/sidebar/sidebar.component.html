Toxicity performs sentiment analysis on reddit comments from a given subreddit, or Twitter posts with a given hashtag. The analysis returns an average of the sentiment scores of each comment/post, and rates them according to this scale:

Non-Toxic (Score ≥ 2)
Very Positive (1 ≤ Score < 2)
Positive (0.25 ≤ Score < 1)
Neutral (-0.25 < Score < 0.25)
Negative (-1 < Score ≤ -0.25)
Very Negative (-2 < Score ≤ -1)
Toxic (Score ≤ -2)
